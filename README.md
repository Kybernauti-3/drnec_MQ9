# MQ-9 a MQ-2
## Senzor na detekci plynů
Použití pro detekci hořlavých plynů, jakými jsou: LPG, butan, metan, alkohol, propan, vodík, atd. Pro domácí nebo průmyslové užití.
## Zapojení
VCC → 3V3 \
GND → GND \
A0 → GP27_A1 \
D0 → GP21
